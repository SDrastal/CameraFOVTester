# CameraFOVTester

## Unreal Engine Version: 5.6.1

## C++ File: CameraFOVModifier (Source -> CameraFOVTester -> Public -> CameraFOVModifier)

    In this C++ file I took an blueprint that I created for my workshop class and remade it in C++ with the same functionality. Since the ability to convert a blueprint into a C++ class was removed in Unreal Engine 5, I wanted to see how difficult it would be to do this myself. The FOV of the player's camera will increase along the player's speed, using interpolation to make the transition smooth and steady. Further descriptions of the code and its functionality can be found within the file itself.
